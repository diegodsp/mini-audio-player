Mini Audio Player
=================
This is a mini audio player for Angular.

## Dependencies
- Angular 8+;
- Fontawesome or Forawesome.

## Example of usage
```html
<mini-audio-player [url]="https://www.soundhelix.com/examples/mp3/SoundHelix-Song-1.mp3"></mini-audio-player>
```
